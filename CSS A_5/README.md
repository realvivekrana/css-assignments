# CSS Animation Projects Collection

Is repository me 3 alag-alag CSS animation projects hain. Har folder ke andar ek unique animation technique dikhaya gaya hai.

---

## Folder 1: Animated Blocks
- CSS keyframe animations like skew, rotate, scale.
- Blocks animate continuously in rows.
- Files: `index.html`, `style1.css`

## Folder 2: Hover Boxes
- Boxes enlarge on hover with smooth CSS transitions.
- Different timing for width and height transitions.
- Files: `index.html`, `style2.css`

## Folder 3: Independence Day Animation
- Animated scene with moving sun, road, city, car with rotating wheels.
- Uses images and layered backgrounds.
- Files: `index.html`, `style3.css`, images required

---

### How to run
- Open the respective `index.html` file inside each folder in any modern browser.
- Make sure CSS files and images (for Folder 3) are present in the same folder.

---

### Navigation
- Each project contains links to navigate between projects (if you open them locally or on a server).
